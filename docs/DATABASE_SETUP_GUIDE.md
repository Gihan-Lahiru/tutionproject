## Render.com Setup (No Credit Card Needed)

### Step 1: Create Render Account

1. Go to **https://render.com**
2. Sign up with email (no card required)
3. Verify email

### Step 2: Create MySQL Database

1. Click **"New +"** → **"MySQL"**
2. Configure:
   - **Name:** `tuition-sir-db`
   - **Database:** `tuition_sir_db`
   - **User:** `tuition_sir_user`
   - **Region:** Singapore (or closest)
   - **Plan:** **Free**
3. Click **"Create Database"**
4. Wait 2-3 minutes

### Step 3: Get Database Credentials

1. Click on your database
2. Go to **"Connect"** tab
3. Copy these details:
   - **Internal Database URL** (use this!)
   - **Host** (e.g., `oregon-postgres.render.com`)
   - **Port** (usually `3306`)
   - **Database** (`tuition_sir_db`)
   - **Username** (e.g., `tuition_sir_user`)
   - **Password** (generated by Render)

### Step 4: Update Backend Environment Variables

Open `backend/.env` and update with Render database credentials:

```env
# Render MySQL Credentials (from Step 3)
DB_HOST=oregon-postgres.render.com
DB_PORT=3306
DB_NAME=tuition_sir_db
DB_USER=tuition_sir_user
DB_PASSWORD=your_render_generated_password
```
